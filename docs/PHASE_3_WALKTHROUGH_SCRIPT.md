## Technical Setup for Recording

### Screen Recording Software
- **Recommended**: OBS Studio (free) or Loom
- **Resolution**: 1920x1080 minimum
- **Frame Rate**: 30fps
- **Audio**: Clear microphone, test audio levels

### Browser Setup
- Use Chrome or Edge for best compatibility
- Clear browser cache before recording
- Have all tabs ready in advance
- Test all functionality before recording

### Recording Tips
1. **Speak clearly and at moderate pace**
2. **Pause briefly between sections**
3. **Highlight important metrics with cursor**
4. **Use zoom for small text/details**
5. **Keep backup of recording**

### Pre-Recording Checklist
- [ ] All systems tested and working
- [ ] Demo data prepared
- [ ] Browser bookmarks set up
- [ ] Audio levels tested
- [ ] Screen resolution optimized
- [ ] Recording software configured
