## ü§ñ **What is the Benefits AI Chatbot?**

The Benefits AI Chatbot is an intelligent assistant that helps employees understand and manage their workplace benefits through natural conversation. Think of it as having a personal benefits specialist available 24/7!

### **What It Does:**
- **Answers Questions**: Ask about health insurance, dental, vision, retirement plans
- **Compares Plans**: Shows side-by-side comparisons of different benefit options
- **Calculates Costs**: Helps you understand out-of-pocket expenses
- **Analyzes Documents**: Upload benefit documents for instant analysis
- **Provides Recommendations**: Suggests the best plans based on your needs

### **Who Can Use It:**
- **Employees**: Access benefits information and get personalized help
- **HR Administrators**: Manage the system and view analytics
- **Company Admins**: Configure settings and monitor usage

## üõ†Ô∏è **System Setup and Deployment**

### **For Technical Teams**

#### **Prerequisites**
- Vercel account (free tier available)
- GoDaddy domain access
- Basic understanding of web hosting

#### **Step 1: Vercel Setup**
1. **Create Vercel Account**
   - Go to [vercel.com](https://vercel.com)
   - Sign up with GitHub account
   - Choose free tier

2. **Import Project**
   - Click "New Project"
   - Import from GitHub repository
   - Select the benefits chatbot repository
   - Choose "Root Directory"

3. **Configure Settings**
   - Framework: Next.js
   - Build Command: `npm run build`
   - Output Directory: `.next`
   - Install Command: `npm install`

#### **Step 2: Domain Configuration (GoDaddy)**
1. **Login to GoDaddy**
   - Go to your domain management
   - Navigate to DNS settings

2. **Add CNAME Record**
   ```
   Type: CNAME
   Name: amerivetaibot
   Value: cname.vercel-dns.com
   TTL: 600 (10 minutes)
   ```

3. **Wait for Propagation**
   - Changes take 10-15 minutes
   - Test by visiting amerivetaibot.bcgenrolls.com

#### **Step 3: Environment Configuration**
Add these environment variables in Vercel:

```bash
# Application
NEXT_PUBLIC_APP_URL=https://amerivetaibot.bcgenrolls.com
NEXT_PUBLIC_DEPLOYMENT_MODE=production
NEXT_PUBLIC_SUBDOMAIN=amerivetaibot

# Authentication
```

## üîß **Troubleshooting**

### **Common Issues and Solutions**

#### **Can't Login**
**Problem**: Login page not working or wrong password
**Solutions**:
1. Check you're using the correct URL: amerivetaibot.bcgenrolls.com
2. Verify you're using the right password:
   - Employees: `amerivet2024!`
   - Admins: `admin2024!`
3. Clear your browser cache
4. Try a different browser
5. Contact support: 888-217-4728

#### **Chat Not Responding**
**Problem**: AI assistant not answering questions
**Solutions**:
1. Check your internet connection
2. Refresh the page
3. Try asking a simpler question
4. Wait a few seconds and try again
5. Contact support if problem persists

#### **Can't Upload Documents**
**Problem**: Document upload not working
**Solutions**:
1. Check file size (must be under 50MB)
2. Verify file type (PDF, Word, text, or image)
3. Try a different file
4. Check your internet connection
5. Contact support for help

#### **Slow Performance**
**Problem**: System running slowly
**Solutions**:
1. Check your internet connection
2. Close other browser tabs
3. Try refreshing the page
4. Clear browser cache
5. Try a different browser

#### **Mobile Issues**
**Problem**: System not working on mobile
**Solutions**:
1. Make sure you're using a modern browser
2. Check your mobile internet connection
3. Try rotating your phone (portrait/landscape)
4. Clear mobile browser cache
5. Contact support for mobile-specific help

### **Error Messages**

#### **"Session Expired"**
**What it means**: You've been logged out automatically
**What to do**: Simply log in again with your password

#### **"Rate Limit Exceeded"**
**What it means**: You're making too many requests too quickly
**What to do**: Wait a few minutes and try again

#### **"File Too Large"**
**What it means**: Your document is bigger than 50MB
**What to do**: Compress the file or split it into smaller parts

#### **"Invalid File Type"**
**What it means**: Your file format isn't supported
**What to do**: Convert to PDF, Word, or text format
