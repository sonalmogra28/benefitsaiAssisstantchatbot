# Benefits AI Chatbot User Guide

## Getting Started

### Access
Navigate to your company's Benefits AI Chatbot URL. Login using your corporate credentials.

### First Login
1. Enter your work email address
2. Complete Azure AD B2C authentication
3. Accept terms of service
4. Set up profile preferences

## Core Features

### Chat Interface
The main chat interface allows direct communication with the AI assistant about benefits questions.

**Starting a Conversation:**
- Type your question in the message box
- Press Enter or click Send
- Wait for AI response (typically 1-3 seconds)

**Example Questions:**
- "What health insurance plans are available?"
- "How much does dental coverage cost?"
- "What's my deductible for vision care?"
- "When is open enrollment?"

### Document Upload
Upload benefit documents for AI analysis and reference.

**Supported Formats:**
- PDF files
- Microsoft Word documents (.doc, .docx)
- Text files (.txt)
- Maximum file size: 50MB

**Upload Process:**
1. Click "Upload Document" button
2. Select file from your device
3. Add title and description
4. Click "Upload"
5. Wait for processing (1-5 minutes)

**Using Uploaded Documents:**
- Ask questions about specific documents
- Reference document content in conversations
- Compare information across documents

### Benefits Comparison
Compare different benefit plans side-by-side.

**Accessing Comparison:**
1. Navigate to "Benefits" section
2. Select "Compare Plans"
3. Choose plans to compare
4. Review detailed comparison table

**Comparison Data:**
- Monthly costs
- Deductibles
- Coverage percentages
- Out-of-pocket maximums
- Network providers

## Troubleshooting

### Common Issues

**Login Problems:**
- Clear browser cache and cookies
- Try incognito/private browsing mode
- Contact IT support for credential issues

**Chat Not Responding:**
- Check internet connection
- Refresh the page
- Try a different browser
- Contact support if problem persists

**Document Upload Fails:**
- Verify file format is supported
- Check file size is under 50MB
- Ensure stable internet connection
- Try uploading one document at a time

**Slow Performance:**
- Close unnecessary browser tabs
- Check internet speed
- Clear browser cache
- Try during off-peak hours

### Error Messages

**"Authentication Required"**
- Log in again with your credentials
- Contact IT if login continues to fail

**"Document Processing Failed"**
- Try uploading the document again
- Check file format and size
- Contact support if problem persists

**"Rate Limit Exceeded"**
- Wait 15 minutes before trying again
- Reduce frequency of requests
- Contact admin if consistently hitting limits
