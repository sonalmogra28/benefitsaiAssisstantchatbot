# Benefits AI Chatbot Admin Guide

## Admin Dashboard Access

### Login Process
1. Navigate to admin portal URL
2. Use admin credentials (Azure AD B2C)
3. Complete multi-factor authentication
4. Access admin dashboard

### Role Permissions
- **Company Admin**: Manage company users and settings
- **Super Admin**: Platform-wide administration
- **Support Admin**: Limited access for troubleshooting

## User Management

### Adding Users
1. Navigate to Users section
2. Click "Add User"
3. Enter user details:
   - Email address
   - Full name
   - Department
   - Role assignment
4. Send invitation email
5. User completes registration

### User Roles
- **Employee**: Basic chat and document access
- **Manager**: Team analytics and reporting
- **HR**: User management and benefit administration
- **Admin**: Full company administration

### Bulk User Operations
- CSV import for multiple users
- Bulk role assignments
- Mass email notifications
- User status updates

### User Monitoring

- Login frequency analysis
- Feature usage statistics
- Performance metrics

## Benefits Management

### Plan Configuration
1. Navigate to Benefits section
2. Click "Add Benefit Plan"
3. Enter plan details:
   - Plan name and type
   - Coverage details
   - Cost information
   - Enrollment periods
4. Configure AI training data
5. Activate plan
